# Quran API

an open-source API providing access to Quranic verses, tafseer, and translations.

## Features

- Fetch verses by surah and ayah.
- Retrieve tafseer and translations.

## Beta

```
https://quran.alile.us/api

/api/suras/
/api/suras/:id

```
