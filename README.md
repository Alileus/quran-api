# Qanun API

**Qanun.io** is an open-source API providing access to Quranic verses, tafseer, and translations.

## Features

- Fetch verses by surah and ayah.
- Retrieve tafseer and translations.

## Beta

```
https://beta.qanun.io/api

/api/suras/
/api/suras/:id

```
